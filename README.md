## NLP project highlight

### Emotion Detection: CNN vs. DistilBERT vs. RoBERTa

I benchmarked three architectures on a six-class emotion-classification task using more than 37,000 training samples after augmentation.

| Model | Test accuracy | Practical trade-off |
|---|---:|---|
| CNN (Conv1D) | 63.52% | Lightweight baseline with fast training |
| DistilBERT | 78.18% | Strong balance between accuracy and deployment cost |
| **RoBERTa** | **80.90%** | Best overall predictive performance |

The project covers preprocessing, class balancing, data augmentation, transformer fine-tuning, confusion-matrix analysis, and model comparison on real text examples.
